## Quick Setup Checklist

### Equipment Needed
- [ ] Screen recording software (ScreenFlow, Camtasia, or OBS)
- [ ] Quality microphone (Blue Yeti or similar)
- [ ] Quiet recording environment
- [ ] Video editing software
- [ ] Demo environment with sample data

### Pre-Recording
- [ ] Review script thoroughly
- [ ] Set up demo environment
- [ ] Prepare sample products/data
- [ ] Test audio levels
- [ ] Close unnecessary applications
- [ ] Set screen resolution to 1920x1080
- [ ] Clear browser history/cache
- [ ] Disable notifications

### Recording Tips
- Speak clearly and at moderate pace
- Pause between sections for easier editing
- Record in segments (easier to fix mistakes)
- Show mouse cursor for clarity
- Zoom in on small UI elements
- Record multiple takes of difficult sections
- Save frequently during recording

### Post-Recording
- [ ] Edit out mistakes and long pauses
- [ ] Add intro/outro
- [ ] Add callouts and annotations
- [ ] Add background music (subtle)
- [ ] Add captions/subtitles
- [ ] Export in 1080p
- [ ] Upload to YouTube/Vimeo
- [ ] Update documentation with video IDs
- [ ] Test embedded videos

## Quality Standards

### Video Quality
- Resolution: 1920x1080 (1080p) minimum
- Frame rate: 30fps or 60fps
- Bitrate: 8-12 Mbps
- Format: MP4 (H.264)

### Audio Quality
- Sample rate: 48kHz
- Bitrate: 192 kbps minimum
- Format: AAC
- No background noise
- Consistent volume levels

### Visual Standards
- Clear, readable text (minimum 16pt)
- High contrast for readability
- Smooth transitions
- Professional callouts
- Brand-consistent colors

## Accessibility Requirements

### Captions
- Accurate closed captions for all videos
- Include sound effect descriptions
- Proper punctuation and timing
- Review auto-generated captions

### Transcripts
- Provide full text transcript
- Include timestamps
- Host on documentation site
- Make downloadable

### Visual Accessibility
- Sufficient color contrast (WCAG AA)
- Don't rely solely on color
- Describe visual elements verbally
- Use large, readable fonts

## Budget Estimate

### Equipment (One-time)
- Microphone: $100-200
- Screen recording software: $0-300
- Video editing software: $0-300
- **Total:** $100-800

### Production (Per Video)
- Script writing: 2-4 hours
- Recording: 1-3 hours
- Editing: 3-6 hours
- Review & revisions: 1-2 hours
- **Total:** 7-15 hours per video

### Full Series (18 Videos)
- **Time:** 126-270 hours

- **Timeline:** 3-6 months (part-time)
