## Browser-Specific Notes

### Chrome/Chromium
- Reference browser for development
- Best DevTools support
- Full ES2020 support

### Firefox
- Excellent standards compliance
- Good performance
- May have slight CSS rendering differences

### Safari/WebKit
- iOS Safari is critical for mobile users
- May require vendor prefixes
- Stricter security policies

### Edge
- Chromium-based (similar to Chrome)
- Good compatibility expected
- Windows-specific testing

### Mobile Browsers
- Touch events vs mouse events
- Smaller viewports
- Performance considerations
- Network conditions vary

## Test Execution

### How to Run Tests

```bash
# Run all browser tests
cd widget
npm run test:browser

# Run specific browser
npm run test:browser:chromium
npm run test:browser:firefox
npm run test:browser:webkit

# Run mobile tests
npm run test:browser:mobile

# Interactive UI mode
npm run test:browser:ui

# View report
npm run test:browser:report
```

### Prerequisites

1. Install dependencies: `npm install`
2. Install Playwright browsers: `npx playwright install`
3. Build widget: `npm run build`
