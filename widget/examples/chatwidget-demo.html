ChatWidget Component Demo

A fully functional chat widget with all features implemented

Implemented Features:

  • Toggle button with smooth animations
  • Responsive chat window (mobile & desktop)
  • Message list with user/assistant/system messages
  • Product recommendations with cards
  • Input box with auto-resize textarea
  • Typing indicator animation
  • Session management and history persistence
  • Customizable theme (colors, fonts, position)
  • Error handling and recovery
  • Add to cart integration callbacks
Note: The chat widget should appear in the bottom-right corner of your screen. Click the blue button to open it and start chatting!
