What is Async Loading? RECOMMENDED

Async (asynchronous) loading means the widget script loads in the background without blocking your page from rendering. This ensures your website loads fast even if the widget takes a moment to load.

This is the same technique used by Google Analytics, Facebook Pixel, Intercom, and other popular third-party scripts.

How It Works

Step 1: The snippet creates a queue for commands

Step 2: It loads the widget script asynchronously

Step 3: Once loaded, it processes all queued commands

Step 4: The widget initializes and appears on your page

All of this happens in the background without blocking your page load!

Advanced Usage

You can call widget methods before it's fully loaded:

// These commands are queued and executed when widget loads
ra('init', { /* config */ });
ra('open');  // Opens widget automatically
ra('sendMessage', 'Hello!');  // Sends a message
