Mobile-Specific Behavior

On Mobile Devices (< 768px):

  • Widget opens in fullscreen mode
  • Larger touch targets for buttons
  • Optimized keyboard handling
  • Swipe down to close
  • Auto-scroll to input when keyboard opens

On Tablets (768px - 1024px):

  • Floating window (400px x 600px)
  • Positioned in corner
  • Can be minimized
  • Touch and mouse support

On Desktop (> 1024px):

  • Compact floating window
  • Positioned in bottom-right corner
  • Keyboard shortcuts supported
  • Hover effects enabled

Touch Gestures

The widget supports intuitive touch gestures on mobile:

  • Swipe Down: Close the widget
  • Tap: Open/close widget, select products
  • Scroll: Navigate through messages
  • Long Press: Copy message text
