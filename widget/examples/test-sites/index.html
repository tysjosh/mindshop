üöÄ How to Run Tests

  1. Make sure you've built the widget: cd widget && npm run build
  2. Start a local server (if needed): npm run serve
  3. Click on any test site card above to open it in a new tab
  4. Test the widget functionality using the provided controls
  5. Check the console output for any errors or warnings
  6. Verify callbacks are working (add to cart, checkout, etc.)
  7. Test on different browsers (Chrome, Firefox, Safari, Edge)
  8. Test on different devices (desktop, tablet, mobile)

‚úÖ Testing Checklist

  • Widget loads without errors
  • Widget appears in correct position
  • Open/close functionality works
  • Messages can be sent
  • Product recommendations appear
  • Add to cart callback fires
  • Checkout callback fires
  • Analytics events are tracked
  • Session management works
  • History can be cleared
  • Session can be reset
  • Widget is responsive on mobile
  • No console errors
  • Performance is acceptable
